## 👋 소개

- **일석4조**는 동시에 네 가지 이득을 본다라는 말이므로, 팀 구성원 모두 이번 팀 프로젝트로 많은 이득을 가져간다는 의미를 가진 팀 이름입니다.
- 우리 팀은 **타워 디펜스 게임**을 제작합니다.
